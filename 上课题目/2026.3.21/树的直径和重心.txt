树的直径：
	无向树,每条边有边权（边权非负） 
	任意两点u,v之间有唯一的路径
	路径长度定义为经过的边的边权和
	最长的路径称为树的直径

	直径不一定唯一,但直径长度唯一

找树的直径：2遍dfs
	第一遍dfs：从任意一个点u出发
		找到距离u最远的点a

	int d[maxn];
	void dfs(int u, int p){
		for(auto v : G[u]) if(v != p){
			d[v] = d[u] + 1;
			dfs(v, u);	
		}
	}

	dfs(u, 0);
	找d数组的最大值对应的下标a

	第二遍dfs：从a出发
		找到距离a最远的点b
	那么a-b就是直径
	
这样的dfs结束后,找到d的最大值对应的下标
	一定是某条直径的一个端点

假设v是从u出发能走到的最远的点,但它不是直径端点 
情况1	u到v的路径 和 a到b的路径（直径）没有交点
		那么一定能找到两个点,把两条路径连起来
		d(u,A) < d(u,v) < d(A,B)
		d(u,B) < d(u,v) < d(A,B)

		d(w,v) > d(w,B)
		d(w,v) > d(w,A)
		距离B最远的应该是谁？
		d(B,v) = d(B,c) + d(c,w) + d(w,v)
			   > d(B,c) + d(c,w) + d(w,A)
			   = d(B,c) + d(c,w) + d(w,c) + d(c,A)
			   = d(B,A) + 2 * d(c,w)
			   > d(B,A)
		矛盾！
情况2：如果两条路径有交点,那么c和w变成同一个点


直径可能有多个,在星形图（n-1个节点是叶子）
	一棵树中,有没有可能出现两条不相交的直径？
	不可能,任意两条直径必定相交 

求有哪些边位于所有的直径上
	把直径拉直,不在直径上的树垂到下面
	把直径看作“挂绳”,不在直径上的子树看作“挂坠”
	每个挂坠有深度,表示从挂绳上的一点u,到挂坠内部的最长距离
	如果u到挂坠内部的最长距离恰好等于u到某个直径端点的距离
		那么说明有一条直径位于挂坠内部

	如果u到挂绳左端点的距离,等于u到挂坠内部最深处的距离
		就说明u的左边不存在所有直径都经过的边

所有直径都一定通过的边是挂绳的一个区间

树的重心
	节点质量：对于节点u,把u从树中删除
		删除后,树会分裂成几个连通块
		其中最大连通块的节点数称为u的质量
	树的重心：质量最小的节点

求树的重心：
	在dfs的过程中,需要知道把u删掉,
	剩下的连通块大小分别是多少

	u的每个子树都是单独的连通块
	还有一个连通块是u上面的连通块

	int m[maxn]; 
	int dfs(int u, int p){
		int C = 1,  		//C来计算u为根的子树大小 
		for(auto v : G[u]){
			int Cv = dfs(v, u); 	//Cv就是v为根的子树大小 
			C += Cv;
			m[u] = max(m[u], Cv);
		}
		m[u] = max(n - C, ans); 		//ans就是u的质量
		return C; 
	}

树的重心最多有2个,如果有2个,那么它们必定相邻
